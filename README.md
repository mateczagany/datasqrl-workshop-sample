# DataSQRL Workshop - Sample Implementations

This repository contains sample implementations to use as a reference in the DataSQRL workshop.
All reference implementations are based on the [workshop data catalog](https://github.com/datasqrl-colab/datasqrl-workshop-catalog).

* [Seller 360](seller360): Analysis orders for seller analytics
* [Seller 360 - RBAC](seller360-rbac): Extends the above implementation with authentication/authorization to provide secure data access control.
